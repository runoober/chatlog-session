# Changelog

## [0.18.0] - 2026-03-10

### Added

#### MessageBubble 点击能力增强

- 为 `MessageBubble` 增加多类型消息点击处理能力，统一处理文件、链接、小程序、位置、图片、视频等消息点击行为。
- 新增转发消息点击弹窗入口，点击转发消息可直接打开聊天记录详情对话框。

### Changed

- 聊天气泡组件的交互逻辑从“仅渲染”升级为“渲染 + 类型化点击分发”，为后续媒体预览和行为收敛打下基础。
- 升级应用版本号到 `0.18.0`。

### Technical Details

- **修改文件**:
  - `src/components/chat/MessageBubble.vue` - 增加统一点击分发与各消息类型点击处理逻辑
  - `package.json` - 版本号更新至 `0.18.0`

### Notes

- 本版本聚焦“消息交互能力补齐”，暂未引入会话级媒体串行浏览。
